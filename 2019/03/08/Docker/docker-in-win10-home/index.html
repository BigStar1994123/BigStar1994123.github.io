所以 Docker Desktop for Windows 無法安裝

只有 Windows 10 Professinal or Enterprise 64-bit 才可以

Home版只能安裝 Docker Toolbox 去模擬 Linux 環境跑 Docker
安裝完畢後跑跑看新手教學
MsDoc: Use Docker
最後執行 Localhost:5000 跑不起來

最後找到解法
必須先在模擬環境內下 docker-machine ip 取得IP位置
先能正確的 Run 起來
